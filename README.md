paper-latest
============
[![Crates.io](https://img.shields.io/crates/v/paper-latest?style=flat-square)](https://crates.io/crates/paper-latest)

Grabs the latest download from PaperMC's download api. Supports all projects that PaperMC offers!

## Installation
You'll need `cargo` to install this. Then simply do `cargo install paper-latest`.
